Box model

O Box Model é um conceito fundamental no CSS que define como os elementos de uma página web são exibidos e manipulados. Cada elemento em uma página é representado como uma caixa retangular, composta por quatro áreas principais:


  • Content (Conteúdo): A área onde o conteúdo (texto, imagem, etc.) é exibido.

  • Padding (Preenchimento): O espaço entre o conteúdo e a borda da caixa.

  • Border (Borda): A borda ao redor da caixa de conteúdo e padding. Pode ser estilizada com cor, espessura e estilo.

  • Margin (Margem): O espaço fora da borda do elemento. Este espaço separa o elemento dos outros ao seu redor e é completamente transparente.

Oque fazer:

  1. Entender a Posição Atual: Observe a posição inicial da imagem da princesa e a imagem fantasma.

  2. Ajustar Margens: Use as caixas de entrada para inserir valores de margin-left e margin-top. Esses valores movem a imagem da princesa para a direita e para baixo, respectivamente.

  3. Visualizar o Resultado: Após inserir os valores, visualize a página para ver como a imagem se move. Ajuste conforme necessário até que a princesa esteja perfeitamente alinhada com a imagem fantasma.
Code

.pincesa-editor {

margin-left: px;

margin-top: px;

max-width: 8vw;

background-color: #1d1e22;

box-shadow: 0px 4px 30px rgba(0, 0, 0, 0.5);

border-radius: 8px;

}
