Display em CSS

A propriedade display em CSS define como um elemento HTML é exibido na página. Os valores principais incluem block, que inicia em uma nova linha e ocupa toda a largura disponível; inline, que não inicia uma nova linha e se ajusta ao conteúdo ao redor; inline-block, combinando características de block e inline; flex, permitindo layouts flexíveis com alinhamento e expansão dos itens dentro de um contêiner; grid, que organiza elementos em uma grade bidimensional; table, transformando elementos em blocos de tabela; e none, que oculta completamente o elemento da renderização. A escolha do valor apropriado depende do layout desejado e do comportamento esperado dos elementos na interface do usuário.

margin-top: 25%;
